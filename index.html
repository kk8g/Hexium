// Installation

Up and running
in minutes

Two files — a tiny loader in Tampermonkey and the full script hosted on GitHub. No build tools, no hassle.

loader.user.js
1

Install Tampermonkey

Get the browser extension from tampermonkey.net — Chrome, Firefox, Edge, and Safari all supported.

2

Create a new script

Open the Tampermonkey dashboard → click the + icon to create a new userscript.

3

Paste the loader

Replace all existing content with the loader script on the left, then save with Ctrl+S.

4

Visit Pekora

Navigate to pekora.zip while logged in — Hexium loads automatically and verifies your access.

// Features

Everything you need
to trade smarter

Built specifically for the Pekora ecosystem with real-time market data from Koromons.

Trade Window

A fully custom trade UI with inventory browsing, search, sorting, and live value analytics powered by Koromons data.

Mass Trader

Blast trades to all owners of a target item, or build a custom target list with specific items per user — fully automated.

Item Monitor

Polls the catalog on a custom interval and fires notifications plus Discord webhooks when new limited items drop.

Trade History

View all inbound and outbound trades with full item details, value diffs, and accept/decline directly from the modal.

Auto Refresher

Clicks the genre selector on a configurable interval and hard-refreshes the page periodically. Hotkey toggled on the fly.

Six Themes

Skeet, Void, Crimson, Arctic, Gold, and Matrix — with accent colours, card glows, and gradient top borders per theme.

Page Customiser

Transparent sidebars, blurred navbars, custom backgrounds, animated usernames, hidden ads, and per-page frame styles.

Watermark HUD

Draggable on-screen overlay showing session time, live ping, and username. Configurable position, scale, and opacity.
